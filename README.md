# Natural Rewriter —— 让你的学术写作更像"你自己写的"

> 代码和论文都需要打磨。这个工具帮你的学术文本更自然、更易读，像自己敲出来的，不是模板生成的。

---

## 这玩意干嘛的？

你写了草稿——不管是自己写的、翻译工具翻的、还是其他方式生成的——这个工具帮你把表达打磨得更自然流畅。

两个模式，一键切换：

| 模式 | 触发词 | 效果 |
|------|--------|------|
| 📝 **学术润色** | `[polish]` | 逻辑更严密、表达更学术，该严谨的地方帮你补上 |
| ✍️ **自然化润色** | `[enhance]` | 消除生硬感和模板腔，让行文更像真人手笔 |

默认不加标签就走学术润色模式。

---

## 怎么用？

把 `SKILL.md` 丢进你的 Claude Code skills 文件夹，然后：

```
[enhance] 基于深度学习的目标检测算法在复杂场景下…
```

没有标签也行，默认帮你润色。

---

## 亮点

- 🌏 **中英文通吃** —— 两种语言各有专门的处理策略，不是粗暴翻译
- 🔒 **术语不动** —— API 路径、代码片段、专有名词全部原样保留
- 🧹 **净输出** —— 只返回改写后的文本，不多说一句
- ✅ **质量自检** —— 内置写作自然度检查清单，输出前自查一遍

---

## 原理一句话

中文走"冗余注入 + 词汇降级 + 口语化"，英文走"英译中 → 中文塑形 → 机械回译"，利用中文语法的模具效应重塑英文表达习惯。
