# Academic Rewriter — 双态学术文本重塑专家

降低 AI 检测率的学术文本改写工具，集成"学术润色"与"去 AI 化"双模式。

## 模式

| 模式 | 标签 | 用途 |
|------|------|------|
| 深度学术润色 | `[polish]` | 保持技术准确，增强逻辑性、解释性 |
| 风格拟态去AI化 | `[enhance]` | 还原人类写作自然感，降低 AI 特征 |

## 安装

将 `SKILL.md` 放入你的 Claude Code skills 目录即可。

## 使用方法

```
[enhance] 你的学术文本...
```

无标签时默认使用 `[polish]` 润色模式。

## 特性

- 支持中文 / 英文双语
- 技术术语严格隔离保护
- 输出纯净改写文本，无多余前缀后缀
- 内置 AI 味道自查清单
